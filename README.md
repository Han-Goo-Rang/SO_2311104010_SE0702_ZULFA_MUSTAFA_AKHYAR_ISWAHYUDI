# Praktikum Sistem Operasi - Xinu OS

Repository ini berisi materi dan dokumentasi praktikum Sistem Operasi menggunakan Xinu OS (eXperimental UNIx). Praktikum ini dirancang untuk memberikan pemahaman mendalam tentang konsep sistem operasi melalui eksplorasi langsung pada source code dan implementasi.

## 🛠️ Tools & Environment

- **VirtualBox**: Virtualisasi untuk menjalankan environment Linux
- **Ubuntu**: Sistem operasi host untuk development
- **Xinu OS**: Sistem operasi eksperimental untuk pembelajaran
- **Sourcetrail**: Tool untuk navigasi dan visualisasi source code

## 📚 Struktur Praktikum

Praktikum ini dibagi menjadi 16 modul mingguan yang mencakup:

### Bagian 1: Pengenalan Xinu (Minggu 1-4)

- **01_Running_Modul**: Menjalankan modul pertama Xinu
- **02_Instalasi_Xinu**: Setup dan instalasi Xinu OS
- **03_Eksplorasi_Xinu**: Eksplorasi dasar sistem Xinu
- **04_Membaca_Source_Code_Xinu**: Analisis source code menggunakan Sourcetrail

### Bagian 2: Manajemen Proses & Konkurensi (Minggu 5-8)

- **05_Eksplorasi_Proses**: Konsep dan implementasi proses
- **06_Sekuensial_dan_Konkuren**: Pemrograman sekuensial vs konkuren
- **07_Semaphore**: Sinkronisasi menggunakan semaphore
- **08_Remedial_CLO_2**: Evaluasi CLO 2

### Bagian 3: System Call & Shell (Minggu 9-11)

- **09_Syscall_Xinu**: System call di Xinu OS
- **10_Shell**: Implementasi shell
- **11_Memori_Xinu**: Manajemen memori di Xinu

### Bagian 4: Linux & Scripting (Minggu 12-16)

- **12_Linux_dan_Windows**: Perbandingan Linux dan Windows
- **13_Perintah_Dasar_Linux**: Command line Linux essentials
- **14_Scripting**: Shell scripting
- **15_Keamanan_Linux**: Security di Linux
- **16_Remedial_CLO_4**: Evaluasi CLO 4

## 🎯 Tujuan Pembelajaran

Praktikum ini bertujuan untuk:

- Memahami arsitektur dan implementasi sistem operasi
- Menganalisis source code sistem operasi secara langsung
- Memahami konsep proses, konkurensi, dan sinkronisasi
- Menguasai system call dan manajemen memori
- Mengembangkan kemampuan Linux administration dan scripting

## 📖 Cara Menggunakan Repository

Setiap folder mingguan berisi file `youkoso.md` yang berisi:

- Materi pembelajaran
- Langkah-langkah praktikum
- Tugas dan latihan
- Referensi tambahan

## 🚀 Getting Started

1. Install VirtualBox dan setup Ubuntu VM
2. Install Xinu OS sesuai panduan di folder `02_Instalasi_Xinu`
3. Install Sourcetrail untuk analisis source code
4. Ikuti modul secara berurutan mulai dari minggu 1

## 📝 Catatan

Repository ini merupakan dokumentasi praktikum yang akan diisi secara bertahap setiap minggunya.
